FAQ

Is The Division 2 down right now?
This unofficial fan-made tracker shows a derived status from a publicly available official Ubisoft service source. Use the visible official source link for the final status check.
How often is this Division 2 server status page updated?
The automated check runs hourly. If the check fails, the page is marked stale instead of pretending the servers are operational.
What is Delta-03 in The Division 2?
Delta-03 is commonly associated with connection, authentication, or service availability problems. Check your connection, Ubisoft Connect, and the official server status source.
Is Raigulus an official Ubisoft server status page?
No. This is an unofficial fan-made tracker. Ubisoft, The Division, and related marks belong to Ubisoft; the official source remains the Ubisoft service/help page linked here.
